# excel-toData
这是一个利用xml制定excel变成我们想要的对应的项目
1.关联jar(poi-3.14-20160307.jar、
poi-ooxml-3.14-20160307.jar、
poi-ooxml-schemas-3.14-20160307.jar 、
xmlbeans-2.6.0.jar和JAXB)

2.用JAXB技术由bean生成对应的xml，由xml制定规则让excel生成我们想要的对象
   excle——>xml——>pojo
